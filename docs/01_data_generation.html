2 Objective

The objective of this repository is to simulate a dataset with few very simple missingness scenarios and evaluate bias based on (a) on which covariates are involved in the missingness generating mechanism and (b) the strength of association between covariates.

The first step involves the generation of a complete dataset with realistic distributions and associations between covariates.

3 Data generation

The desired distributions as observed in the dataset.

n_total <- 10000

# observed distributions from data

## expoure
X_prevalence <- 0.68

## Z2 (serum creatinine) in exposed and unexposed
Z2_mean_exposed <- 0.85
Z2_sd_exposed <- 0.23

Z2_mean_unexposed <- 0.88
Z2_sd_unexposed <- 0.23

## U (glaucoma or cataract)
U_unexposed <- 0.11
U_exposed <- 0.20
  • Desired overall sample size: 10^{4}

3.1 Generate exposure and covariates

Code 
# Set the seed
set.seed(42)

# Build the total cohort
cohort_X_Z2_U <- tibble(
    X = rbinom(n = n_total, size = 1, prob = X_prevalence),
    Z2 = ifelse(
      X == 1, 
      rnorm(n = n_total, mean = Z2_mean_exposed, sd = Z2_sd_exposed),
      rnorm(n = n_total, mean = Z2_mean_unexposed, sd = Z2_sd_unexposed)
      ),
    U = ifelse(
      X == 1, 
      rbinom(n = n_total, size = 1, prob = U_exposed),
      rbinom(n = n_total, size = 1, prob = U_unexposed)
      ),
    )

3.2 Generate outcome

Generate the outcome with a desired event rate of ~40 events/1,000 person-years of follow-up.

Code 
betas_os <- c(
  X = log(1),
  Z2 = log(5.16),
  U = log(1.48)
  )

set.seed(42)
cohort <- cohort_X_Z2_U |> 
  bind_cols(
    simsurv(
      dist = "exponential",
      lambdas = 0.0087,
      betas = betas_os,
      x = cohort_X_Z2_U,
      maxt = 10
      )
    ) |>  
  select(-id)

# event-rate per 1000 person-years of follow-up
# in original data: ~40 events per 1000 person-years
event_rate_per_1000py <- round(sum(cohort$status)/sum(cohort$eventtime)*1000, 2)
  • Resulting/simulated event rate per 1,000 person-years: 40.45
